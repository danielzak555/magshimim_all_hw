# Ex10 - STL

Link to ex10 google drive folder: <a href="https://drive.google.com/drive/u/0/folders/1W3J1vMeWgvJEguqVn_ek50lkPTWKvfO0/" target="_blank">Ex10 Google Drive</a>

The 10th exercise in 'Advanced Programming' course. <br />
In this exercise we'll build a small Shopping Market system that will be composed out of Items, Stores and Customers. <br />
We will use C++ STL library throughout our exercise classes, and practice: <br />
- Iterators
- Containers (std::set, std::map, std::deque)
- Algorithms (std::sort, std::copy, std::copy_if)
  
The purpose of this exercise is to get familiar with STL and its components.
